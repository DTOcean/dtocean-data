Surging Barge Point Absorber

General Description

The device used in this example is a barge only allowed to surge.

Geometry

The barge dimensions are 7.85x10x10 m.

The mesh of the submerged part of the barge is shown in the figure below.

The mesh is composed of 250 panels.

Body description

The geometric centre of the barge, which is taken as the centre of gravity too,

is set to be the (0, 0, -5) with respect to the global reference system.

Mass

The mass of the barge is 785000 kg.

PTO

The PTO damping is assumed to be 444000 N s/m.

Mooring

The mooring stiffness is assumed to be 1402000 N/m.
