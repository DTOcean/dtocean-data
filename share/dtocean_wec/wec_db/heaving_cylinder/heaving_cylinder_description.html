Heaving Cylinder Point Absorber

General Description

The device used in this example is a cylinder only allowed to heave.

Geometry

The cylinder is a 5 m radius and 10 m draught cylinder.

The mesh of the submerged part of the cylinder is shown in the figure below.

The mesh is composed of 544 panels.

Body description

The geometric centre of the cylinder, which is taken as the centre of gravity too,

is set to be the (0, 0, -5) with respect to the global reference system.

Mass

The mass of the cylinder is 785398.16 kg.

PTO

The PTO damping is assumed to be 159900 N s/m.

Mooring

No mooring stiffness is taken into account.
